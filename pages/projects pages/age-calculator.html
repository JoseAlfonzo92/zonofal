Age Calculator App

A responsive age calculator built for a Frontend Mentor challenge. It features semantic HTML, custom form validation, and animated result output using vanilla JavaScript. Styled with Poppins font, soft shadows, and a playful layout, the app calculates age in years, months, and days based on user input.

Detailed Project Description

Objectives

Build a clean and responsive age calculator that validates user input and displays results with smooth animation. The project needed to follow accessibility best practices and match the design brief provided by Frontend Mentor.

Challenges Faced

Implementing accurate date validation across edge cases, handling leap years, and animating the result output without performance issues. Ensuring the layout remained consistent across screen sizes and devices was also key.

Solutions Implemented

Used semantic HTML5 and custom CSS for layout and styling. Added JavaScript logic for input validation and age calculation, including leap year handling. Applied smooth animations to result output and ensured full keyboard accessibility.

Project Highlights

  • Accurate age calculation with leap year support
  • Custom form validation with error feedback
  • Animated result output for enhanced UX
  • Responsive layout with playful styling
